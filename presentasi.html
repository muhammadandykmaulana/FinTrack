Snippet Kode Utama

script.js - Logika Simpan Transaksi 
function saveTransaction() {
    // 1. Ambil nilai dari input HTML
    const desc = document.getElementById('inp-desc').value;
    const amount = document.getElementById('inp-amount').value;
    const date = document.getElementById('inp-date').value;
    const categoryEl = document.querySelector('input[name="category"]:checked');
    const category = categoryEl ? categoryEl.value : 'Lainnya';

    // 2. Validasi sederhana
    if (!desc || !amount) {
        alert("Deskripsi dan Jumlah wajib diisi!");
        return;
    }

    // 3. Bentuk object transaksi
    const payload = {
        id: state.editingId || Date.now(),
        desc,
        amount: parseInt(amount),
        type: state.transactionType,
        date,
        category
    };

    // 4. Mode edit vs tambah baru
    if (state.editingId) {
        state.transactions = state.transactions.map(t => 
            t.id === state.editingId ? payload : t
        );
        alert("Transaksi Diupdate!");
    } else {
        state.transactions.unshift(payload);
        alert("Transaksi Disimpan!");
    }

    // 5. Simpan ke localStorage + refresh tampilan
    saveData();
    resetForm();
    renderTransactions();
    calculateTotal();
}

Kode ini menunjukkan logika pengambilan data dari DOM, validasi, manipulasi Array objek untuk menyimpan transaksi baru, dan saveData ke Localstorage.

Refleksi & Tantangan

Tantangan Utama

  • Responsive Layout: Sebagai pemula, mengatur tampilan agar rapi di layar HP (Vertical) dan Laptop (Horizontal) cukup sulit.
    Solusi: Menggunakan class Tailwind seperti `flex-col` untuk HP dan `md:flex-row` untuk Laptop.
  • Tailwind CSS: Belajar menggunakan utility classes (`w-full`, `p-4`, `flex`) yang mempercepat styling dibanding CSS biasa.
  • Logika JavaScript: Menghubungkan tampilan HTML dengan data di JS (DOM Manipulation) agar data tampil dinamis tanpa refresh.

Hal Baru yang Dipelajari

  • Implementasi Localstorage Sederhana: Data state login, transaksi dan dompet akan tersimpan di browser user. Data tidak hilang bila direfresh.
  • State Management Sederhana: Memahami cara menampung data dalam variabel Object/Array di Javascript sebelum ditampilkan ke layar.
  • Konsep SPA Sederhana (tanpa framework): Membuat aplikasi terasa cepat dengan menyembunyikan/menampilkan pada active section (`display: none/flex`) alih-alih pindah halaman.
